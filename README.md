# utools-plugin-keepass

在[uTools 插件 - 密码管理器](https://github.com/uTools-Labs/utools-upassword/) 基础上更改为支持 keepass 的密码管理器
目前只能查看，不能修改！

> 使用 react 18、vite 构建
